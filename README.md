Steps to get it to work (on Linux):
1. git clone https://github.com/saurabh-pandey/mesh-wireframe.git
2. cd mesh-wireframe
3. python3 -m venv venv/
4. source venv/bin/activate
5. pip install -r requirements.txt

Now try:
python3 cone.py
